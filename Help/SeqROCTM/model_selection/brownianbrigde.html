brownianbrigde

PURPOSE

Generate a Brownian Bridge of dimension N.

SYNOPSIS

B = brownianbrigde(N)

DESCRIPTION 

BROWNIANBRIDGE Generate a Brownian Bridge of dimension N.

 Input
   N : dimension of the Brownian bridge

 Output
   B : Brownian bridge

 Usage:
   B = brownianbrigde(100);
