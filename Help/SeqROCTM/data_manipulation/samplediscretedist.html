samplediscretedist

PURPOSE

Sample from a discrete distribution.

SYNOPSIS

[xsampled, indices] = samplediscretedist(x, prob, nsamples)

DESCRIPTION 

SAMPLEDISCRETEDIST Sample from a discrete distribution.

 Inputs
   x         : domain of the distribution 
   prob      : probability associated to each value on x
   nsamples  : number of sampled values

 Outputs
   xsampled  : sampled values
   idx       : index of the sampled values
