contextfunction

PURPOSE

Return the context associated to a given (past) sequence.

SYNOPSIS

[c, idx] = contextfunction(past, contexts)

DESCRIPTION 

CONTEXTFUNCTION Return the context associated to a given (past) sequence.
 
 Inputs

     past      : sequence
     contexts  : set of contexts (context tree)
 
 Outputs

     c         : context associated to the sequence [past]
     idx       : index (position) of the context [c] in [contexts]. If do
               not exist, idx=-1

 Usage
            ctxs = {0, [0 1], [1 1], 2};
            past = [0 1 1 0 1 2 0 1];
            [c, idx] = contextfunction(past, ctxs);
