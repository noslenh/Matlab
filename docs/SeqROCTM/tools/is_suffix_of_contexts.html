is_suffix_of_contexts

PURPOSE

Get all the contexts from which a given sequence is a suffix.

SYNOPSIS

[ctxs, index] = is_suffix_of_contexts(ss, contexts)

DESCRIPTION 

IS_SUFFIX_OF_CONTEXTS Get all the contexts from which a given sequence is a suffix.

 Inputs

     ss        : sequence
     contexts  : set of contexts

 Output

     ctxs      : contexts from which ss is a suffix
     index     : indexes of such contexts in the array contexts
