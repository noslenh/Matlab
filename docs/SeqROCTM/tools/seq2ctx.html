seq2ctx

PURPOSE

Converts a sequence in the corresponding sequence of indices of contexts.

SYNOPSIS

ctx = seq2ctx(X, tau)

DESCRIPTION 

SEG2CTX Converts a sequence in the corresponding sequence of indices of contexts.

 INPUT
       X   : a sequence of symbols (context tree model)
       tau : context tree compatible with X

 OUTPUT

       ctx : sequence with the indices of the contexts
